⚙️ CONFIGURAR FIREBASE
1. Crear proyecto Firebase (gratis)
Andá a console.firebase.google.com → "Agregar proyecto" → Crear
2. Activar Realtime Database
Menú izquierdo → Compilación → Realtime Database → Crear → Modo prueba → Habilitar
3. Copiar configuración
⚙️ Configuración del proyecto → Tus apps → Web → copiar firebaseConfig
4. Pegar en este archivo
Abrí el HTML con Bloc de notas y reemplazá los valores de firebaseConfig
